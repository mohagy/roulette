Roulette TV Display Sharing & Embedding

This page demonstrates how to share the Roulette TV Display with multiple viewers in a view-only mode.

Key Features:
  • Real-time synchronization with the main roulette system
  • View-only mode prevents any interaction or modification
  • Automatic updates of draw numbers and countdown timer
  • Responsive design works on all devices
  • Fullscreen mode for optimal viewing
  • Easy embedding on any website

Embedding the Roulette TV Display

Example 1: Basic Responsive Embedding

Add this script tag to your HTML for a responsive embed that works on all devices:

<script src="http://localhost/slipp/tvdisplay/embed.js"></script>

Result:

Example 2: Fixed Size Embedding

You can customize the size and disable responsive behavior:

<script src="http://localhost/slipp/tvdisplay/embed.js" data-width="800px" data-height="400px" data-responsive="false"></script>

Result:

Example 3: Custom Title

You can set a custom title for accessibility:

<script src="http://localhost/slipp/tvdisplay/embed.js" data-title="Live Roulette Results"></script>

Result:

Note: The embedded display auto-refreshes every 5 seconds. The manual refresh parameter (data-refresh) is no longer recommended as it may cause flickering.

Secure Sharing Options

Token-Based Authentication

For restricted access, you can enable token-based authentication:

  1. Edit the share.php file and set $enableAuth = true;
  2. Change the $validToken value to a secure random string
  3. Share the URL with the token parameter:
http://localhost/slipp/tvdisplay/share.php?token=your-secret-token

Access Logging

All access to the share.php page is logged in the logs/view_access.log file, including:

  • Timestamp of access
  • IP address of viewer
  • User agent (browser information)
Important: For production use, consider implementing additional security measures such as HTTPS, IP restrictions, or user authentication.
